## Next.js 14 SEO implementatie
```typescript
// Metadata API (App Router)
export async function generateMetadata(): Promise<Metadata> {
  return {
    title: "<primair zoekwoord> | VorstersNV",   // max 60 tekens
    description: "<160 tekens, actief, zoekwoord>",
    openGraph: { images: [{ width: 1200, height: 630 }] },
    alternates: { canonical: "/shop/<slug>" },
  };
}
// Structured Data JSON-LD (Product schema)
// Sitemap: app/sitemap.ts — ISR met revalidate
// robots.ts — blokkeer /dashboard/ en /api/
```

## SEO-score berekeningsmethode (0-100)
- Title aanwezig + correct formaat: +15
- Meta description aanwezig + juiste lengte: +15
- H1 bevat primair zoekwoord: +10
- Canonical URL ingesteld: +10
- Afbeelding alt-teksten aanwezig: +10
- Structured data aanwezig en geldig: +15
- Laadtijd < 2.5s (LCP): +10
- Interne links aanwezig (min 2): +10
- Hreflang correct: +5
