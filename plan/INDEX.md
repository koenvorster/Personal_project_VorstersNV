## 💡 Technology Stack

### Core Services
```
Frontend:       Next.js 16 (SSR)
Mobile:         React Native
API:            FastAPI (Python)
Real-time:      WebSockets
AI/Agents:      MCP + Claude API
```

### Data & Storage
```
Database:       PostgreSQL 16 (multi-tenant, RLS)
Document DB:    MongoDB
Vector DB:      Pinecone / Weaviate
Cache:          Redis Cluster
Message Queue:  RabbitMQ / Redis Streams
Object Storage: S3 / Google Cloud Storage
```

### Infrastructure
```
Orchestration:  Kubernetes (EKS/AKS/GKE)
IaC:            Terraform
CD:             ArgoCD
Monitoring:     Prometheus + Grafana
Logging:        ELK Stack / Datadog
Service Mesh:   Istio
CDN:            CloudFlare / AWS CloudFront
```
