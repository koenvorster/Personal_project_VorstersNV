1. PROJECTOVERZICHT
────────────────────
VorstersNV is een freelance IT/AI-consultancy platform voor Belgische KMOs.

Stack:
  Frontend    Next.js 16.2 (TypeScript, App Router, Tailwind)   frontend/
  Backend     FastAPI (Python 3.12), SQLAlchemy async, Alembic   api/
  Lokale AI   Ollama — llama3, mistral (47 modules)              ollama/
  AI Agents   32 YAML-definities + agent_runner.py               agents/
  Database    PostgreSQL 16 + Redis 7                            Docker (WSL)
  Betalingen  Mollie webhooks                                    api/routers/betalingen.py
  Auth        Keycloak (dev)                                     Docker
  CI/CD       GitHub Actions                                     .github/workflows/

Huidige architectuur:
  - 49 Claude agents        (.claude/agents/)
  - 32 Ollama runtime agents (agents/)
  - 47 ollama/ modules      (Control Plane, RAG, Knowledge Graph, Compliance
                             Engine GDPR/NIS2/BTW, DiagramRenderer, ReasoningLogger)
  - 13 FastAPI routers      (portal, feedback, streaming, ai_platform, leads...)
  - 15 databasetabellen     (Alembic up-to-date)
  - Consultancy tooling     (scripts/analyse_project.py, code_analyse_agent,
                             consultancy_orchestrator)
